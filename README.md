# 🌐 Networking Basics Lab

🚀 This repository contains practical networking labs created using Cisco Packet Tracer.
It demonstrates fundamental networking concepts including IP addressing, switching, and routing.

---

## 📌 Features

* 📡 IP Address Configuration
* 🔀 Basic Switching (LAN Setup)
* 🌍 Routing Between Networks
* 🧪 Real Lab Simulations

---

## 🛠 Tools Used

* Cisco Packet Tracer
* Command Prompt (Windows)

---

## 📂 Project Structure

```
networking-basics-lab/
│
├── labs/
├── packet-tracer-files/
├── screenshots/
└── README.md
```

---

## 🧪 Labs Included

### 🔹 Lab 1: IP Addressing

* Assigned static IP addresses
* Tested connectivity using ping

### 🔹 Lab 2: Basic Network Topology

* Connected devices using a switch
* Verified LAN communication

### 🔹 Lab 3: Router Configuration

* Connected two different networks
* Configured router interfaces
* Tested inter-network communication
